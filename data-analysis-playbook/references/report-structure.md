# Report Structure

## Executive Summary

- Decision or question
- Main finding
- Business impact
- Recommended next action

## Data Scope

- Source files or tables
- Time range
- Filters
- Row count
- Known exclusions

## Analysis

- Metric definition
- Trend or comparison
- Segment breakdown
- Outliers and data quality notes

## Recommendation

- Action
- Owner
- Timing
- Expected impact
- Follow-up measurement
