# Template catalog

Use this catalog to narrow a presentation brief to three candidates. Match tone before industry; a requested light or dark scheme is a hard constraint. Upstream template files are fetched on demand from `zarazhangrui/beautiful-html-templates` rather than stored here.

## 中文选型速查

先看中文需求里的“使用场景”和“气质词”，再看英文表格的 `best fit` 和 `scheme`：

- 稳重正式：优先 `Signal`、`Blue Professional`、`Monochrome`、`Cartesian`。
- 研究/课题/白皮书：优先 `Monochrome`、`Cartesian`、`Cobalt Grid`、`Vellum`。
- 创意发布/品牌提案：优先 `Creative Mode`、`Studio`、`Bold Poster`、`Editorial Tri-Tone`。
- 轻松培训/共创工作坊：优先 `Daisy Days`、`Capsule`、`Playful`、`Scatterbrain`。
- 数据较密：优先 `Neo-Grid Bold`、`Raw Grid`、`Monochrome`、`Signal`。

## Selection guardrails

- Prefer `Signal`, `Monochrome`, `Blue Professional`, or `Cartesian` for institutional credibility; avoid the overtly playful and poster-led choices.
- Prefer `Neo-Grid Bold`, `Raw Grid`, or `Monochrome` for dense statistics and comparisons.
- Prefer `Daisy Days`, `Capsule`, `Playful`, or `Scatterbrain` for warmth and informality.
- Avoid mixing layouts, typography, colours, or decorations across templates. If a needed layout is absent, create it using the chosen template's own system.
