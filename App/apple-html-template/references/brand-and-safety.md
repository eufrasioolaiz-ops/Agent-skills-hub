# Brand And Safety

This skill was independently authored after reviewing the public `apple-design` Skill in `majiayu000/claude-skill-registry` as a design reference. The source is not bundled or executed.

Source: https://github.com/majiayu000/claude-skill-registry/tree/main/skills/design/apple-design

## Use Boundaries

- Treat "Apple-inspired" as a shorthand for minimal hierarchy, system typography, generous whitespace, restrained motion, and translucent materials.
- Do not use Apple logos, product names, copyrighted page copy, official product photography, proprietary fonts, or a close reconstruction of an Apple page.
- Use only user-supplied, licensed, or newly generated bitmap assets. The bundled Hero bitmap was generated for this template and contains no brand marks or text.
- Do not add tracking, analytics, remote fonts, remote images, credentials, authentication, or network calls to a generated page unless the current user explicitly asks for them.
